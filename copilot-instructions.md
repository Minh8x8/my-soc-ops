# Copilot Instructions

## Design Guide: Anime Bubble Aesthetic

### Overview

This guide defines the visual direction for the Anime Bubble Aesthetic redesign. All UI elements should evoke playful, vibrant, and whimsical anime-inspired bubble motifs. Prioritize rounded shapes, soft gradients, pastel colors, and subtle animations.

### Key Principles

- **Bubble Shapes:** Use rounded corners, pill-shaped buttons, and circular elements throughout.
- **Anime Inspiration:** Incorporate pastel palettes, vibrant gradients, and playful iconography reminiscent of anime visuals.
- **Soft Gradients:** Apply gentle, multi-tone gradients to backgrounds, cards, and UI surfaces.
- **Whimsical Details:** Add sparkles, floating bubbles, and subtle decorative accents.
- **Subtle Animations:** Use pop, bounce, and fade effects for interactive elements.
- **Accessibility:** Ensure high contrast and readable text, even with playful colors.
- **Responsiveness:** All designs must adapt gracefully to different screen sizes.

### Color Palette

- Pastel pink, blue, purple, mint, yellow
- Vibrant accent gradients (e.g., pink-to-blue, purple-to-mint)
- White and soft off-white backgrounds

### Typography

- Rounded, friendly sans-serif fonts (e.g., Nunito, Quicksand, Comic Neue)
- Large, expressive headings
- Playful, readable body text

### Example Elements

- Bubble buttons: pill-shaped, gradient backgrounds, soft drop shadows
- Modal windows: rounded, floating, with sparkles or bubble accents
- Boards/cards: soft edges, pastel backgrounds, animated bubbles

### Assets

- Use anime bubble icons, sparkles, and decorative SVGs
- Backgrounds with floating bubbles or soft gradients

### Implementation Tips

- Leverage Tailwind CSS v4 for rapid, creative styling
- Avoid generic AI-generated looks; reference anime and bubble motifs for inspiration
- Test for accessibility and responsiveness

---

Follow this guide for all new UI work and when refactoring existing components. For questions or creative direction, consult the design team or refer to anime/bubble visual references.
