# Workspace Instructions

## Mandatory Development Checklist

- [ ] Lint: `npm run lint`
- [ ] Build: `npm run build`
- [ ] Test: `npm test`

## Quick Start

1. Install dependencies:
   npm install
2. Start dev server:
   npm run dev
   Open http://localhost:5173

## Structure

- src/: App code (components, logic, data)
- docs/: Documentation
- workshop/: Step guides (00-overview.md → 05-complete.md)

## Tips

- Use workshop for step-by-step learning.
- For help, see docs or ask your team.
